## ðŸ§­ Navigation

### Sidebar

The sidebar provides access to all modules:

- **Dashboard**: Home page with overview
- **Finance**: Financial management
- **HR**: Human resources and payroll
- **SRM**: Supplier relationship management
- **PM**: Project management
- **Warehouses**: Inventory and warehouse operations

### Searching

Use the search box in the sidebar to quickly find menu items:
1. Click the search box
2. Type your query
3. Matching items will be highlighted
4. Click to navigate

### Back to Home

When inside a module, click the "Back to Home" button in the header to return to the dashboard.
